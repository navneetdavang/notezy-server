## Requirement:
- Node.js >= v20.18.0
- Yarn >= 122.22

## Installation:
- Clone Repositroy:
```
git clone https://github.com/navneetdavang/notezy-server.git
```
- To install dependencies:
```
yarn
```

## Run Backend Server:
- Configure envirnoment variables in `.environment` file
- To run server in developement mode:
```
yarn dev
```
- To run server in production mode:
```
yarn prod
```
